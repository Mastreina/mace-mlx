# fp16 特征路径误差评估：结果报告

> agent 被会话限额打断前已跑完全部数值实验（9 个日志 +
> team_fp16_results.json）；本报告由主会话依据其数据撰写。
> 速度背景（主会话另测）：fp16 端到端 1.45-1.52×。

## TL;DR：「误差是否强烈」——分场景，两真两假

担心为真的场景：**高应变/远离平衡构型的绝对能量**（应变 SiO2 实测
-10.4 meV/atom 系统偏移）与**二阶导数类**（小位移有限差分力常数误差
12%，声子不可用）。担心为假的场景：**MD**（力相对误差 <1%，NVT 统计
与 fp32 一致，NVE 无系统性加热）与**常规几何弛豫**（fmax≥0.01 步数
与结果同 fp32 无差别）。

## 1. 单点误差矩阵（fp16 vs fp32，medium-mpa-0 除注明外）

| 体系 | dE meV/atom | F 相对 RMS | maxdF eV/Å |
|---|---:|---:|---:|
| Water81 (H2O 团簇) | +0.87 | 1.00% | 0.035 |
| Si216 rattled | +0.90 | 0.67% | 0.020 |
| Si1000 rattled | +0.86 | 0.62% | 0.025 |
| NaCl216 rattled | +0.12 | 0.96% | 0.005 |
| Cu256 fcc rattled | +1.02 | 0.75% | 0.013 |
| SiO2 cristobalite（未应变） | +1.27 | 0.61% | 0.081 |
| **SiO2 应变构型** | **-10.4** | 0.44% | 0.118 |
| Si216 @ small 模型 | -0.16 | 0.45% | 0.010 |
| Si216 @ large 模型 | +2.18 | 1.07% | 0.017 |

要点：常规构型 ~1 meV/atom、力相对误差 ≤1%。应变 SiO2 的 -10.4 是
**构型相关系统偏移**（未应变同化学只有 +1.3；按元素分解 O -12.3、
Si -8.4，两者同向），且力误差比例不升（0.44%）——影响绝对能量比较
（如凸包、相图），对力驱动的用途影响小。large 模型误差约 2× medium。

## 2. NVE（Si216，1 fs，500 步；关键设计：fp16 轨迹用 fp32 复评能量）

- fp32 基线：漂移 0.000 meV/atom/ps，波动 std 0.004 meV/atom
- fp16（真实漂移，fp32 复评，两个种子）：**-0.26 / +0.13 meV/atom/ps**
  ——符号随种子翻转，是随机游走不是系统性加热；波动 std 0.010-0.013
- 结论：10 ps 量级 NVE 能量不确定度 ~1-3 meV/atom，短轨迹可用；
  百 ps 以上严肃 NVE 守恒研究不建议。

## 3. NVT Langevin（Si216，500 步）

T = 300.6±11.5 K vs fp32 300.5±11.4 K；<Epot> 差 +0.88 meV/atom。
**统计一致，NVT/NPT 生产可用**（恒温器吸收力噪声）。

## 4. 几何弛豫（Si64 rattled，BFGS）

- 收敛步数 fmax=0.05/0.03/0.02/0.01：fp32 [15,18,19,23] vs
  fp16 [15,18,22,25]——基本无差
- 终构型：dE=0.000 meV/atom、RMSD 0.0004 Å；fp16 终构型的真实残余力
  （fp32 评估）0.0068 eV/Å，甚至低于 fp32 跑出的 0.0098
- 紧收敛 fmax=0.002：fp32 9 步；fp16 23 步擦线收敛（last30 中位
  0.0069 在阈值上方抖动）——**fp16 弛豫的实用下限 fmax≈0.005-0.01**

## 5. 二阶导数（Si64 有限差分力常数）

位移 h=0.005 Å 时 fp16 力常数列相对误差 12.4%（h=0.01: 6.0%、
h=0.03: 1.3%，但大 h 引入非谐污染 ~3.5%）。**声子/Hessian/弹性常数
fp16 不可用**——力噪声 ~0.007 eV/Å 除以小位移直接进入二阶导。

## 6. 结论矩阵

| 场景 | fp16 | 依据 |
|---|---|---|
| 单点能量（近平衡） | 可用 | ~1 meV/atom |
| 单点能量（高应变/绝对能量比较） | 慎用 | 实测至 -10 meV/atom 偏移 |
| MD-NVT/NPT | **可用** | 统计与 fp32 一致 |
| MD-NVE ≤10 ps | 可用 | 随机漂移 ~0.2 meV/atom/ps |
| MD-NVE 长轨迹 | 慎用 | 漂移随机游走累积 |
| 弛豫 fmax≥0.01 | **可用** | 步数/结果与 fp32 无差 |
| 弛豫 fmax<0.005 | 不可用 | 0.002 擦线，噪声地板 ~0.007 eV/Å |
| 声子/Hessian/弹性 | **不可用** | 小位移误差 12% |

建议：默认 dtype 保持 fp32；文档把本矩阵写进 README/docstring，
fp16 定位为「MD 与粗弛豫的 1.5× 加速选项」。
